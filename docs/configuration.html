The Performa Modpack is designed for maximum performance, but there are additional settings you can adjust to further optimize your experience.

1. Graphics Settings

  • V-Sync: Disabling V-Sync can improve FPS, but may cause screen tearing. To disable it, go to Options ‚Üí Video Settings and turn off V-Sync.
  • Graphics Quality: In the Video Settings menu, set this to Fast for higher FPS.
  • Render Distance: Recommended 6-8 chunks for best performance.

2. OptiFine Settings (if installed)

If you are using OptiFine, try:

  • Smooth Lighting: Off for FPS boost.
  • Performance: Max FPS.
  • Chunk Loading: Multi-Core.

3. Memory Allocation

To allocate more RAM:

  • Open Minecraft Launcher.
  • Edit your modpack profile.
  • Under JVM Arguments, set -Xmx value (e.g., -Xmx4G).
