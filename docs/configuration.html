The Performa Modpack is designed for maximum performance, but there are additional settings you can adjust to further optimize your experience.

1. Graphics Settings

  • V-Sync: Disabling V-Sync can improve FPS, but may cause screen tearing. To disable it, go to Options -> Video Settings and turn off V-Sync.
  • Graphics Quality: In the Video Settings menu, you can lower the graphics quality for better performance. We recommend setting this to Fast for higher FPS.
  • Render Distance: Reducing your render distance will significantly improve performance, especially on low-end systems. We recommend a render distance of 6-8 chunks for the best performance.

2. OptiFine Settings (if installed)

If you are using OptiFine for additional optimizations, the following settings can improve performance:

  • Smooth Lighting: Turn this off for a noticeable FPS boost.
  • Performance: Set to Max FPS for the best performance.
  • Chunk Loading: Set to Multi-Core for faster chunk loading times.

3. Memory Allocation

Allocating more RAM to Minecraft can help with performance, especially on systems with more than 8GB of RAM. Here's how to do it:

  • Open the Minecraft Launcher.
  • Go to Installations and edit your modpack profile.
  • Click on More Options.
  • Under JVM Arguments, set the -Xmx value (e.g., -Xmx4G for 4GB of RAM).

üí° Tip: It's recommended to allocate 2-4GB of RAM for best performance.

If you're experiencing issues with configuration settings, here are some tips:

  • Low FPS on High Settings: Try lowering your render distance and disabling V-Sync.
  • Game Crashes on Start: Check the logs for mod conflicts, and ensure that your RAM allocation is appropriate.
  • Lag During World Loading: Try reducing your world generation settings or disabling certain resource-heavy mods.
